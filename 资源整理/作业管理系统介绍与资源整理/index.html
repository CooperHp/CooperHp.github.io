作业管理系统介绍与资源整理

发表于 2021-06-26 分类于 资源整理

简介

在超算和集群中通常要使用作业管理系统，关于其介绍见维基百科，最常用的开源作业管理系统包括SLURM (Simple Linux Utility for Resource Management)和PBS (Portable Batch System)。网上关于这两个软件的资源很多，一般使用的话也不用了解太详细，以下仅整理一些资源可供参考学习。

SLURM

官方资源：中文官网、英文官网手册 网络资源：曙光SLURM使用基础教程、博客1、博客2、博客3、博客4 用户手册：百度云盘（密码：u34s）

PBS

官方资源：OpenPBS 官网、OpenPBS GitHub、OpenPBS Public Home 网络资源：博客1、博客2、命令介绍
