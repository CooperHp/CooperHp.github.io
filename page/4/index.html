主要功能

  • 自动化数据可视化和分析
  • 将OVITO数据I/O、分析、渲染等自定义为工作流或python程序

OVITO Pro版本可以根据用户的操作自动生成程序脚本。

Git及其常用命令

发表于 2021-05-25 分类于 编程相关

官方文档、官方Book（中文）

典型工作流程

工作内容：测试修改不同的程序功能并发布。

  1. 指针位于主分支master
  2. 为修改程序功能创建并切换至新分支：git checkout -b dev
  3. 修改程序并提交：git add *；git commit -m "dev"
  4. 切换至主分支：git checkout master
  5. 将程序功能1的分支合并到主分支：git merge dev
  6. 删除程序功能1分支：git branch -d dev
