OVITO module介绍

发表于 2021-06-08 分类于 科研

OVITO module官方手册

主要功能

  • 自动化数据可视化和分析
  • 将OVITO数据I/O、分析、渲染等自定义为工作流或python程序
  • 通过开发新的modifiers或viewport layers扩展OVITO的功能到图形界面

提供的接口包括内部数据模型、数据流框架和渲染功能。常见使用场景是将图形界面的用户复杂操作自动化为程序。

OVITO Pro版本可以根据用户的操作自动生成程序脚本。

OVITO Pro下载自动包含ovito module，不需要license。而图形界面需要license

脚本执行

终端执行

ovitos是预定义的python解释器，与普通的python解释器类似，可通过在命令行直接键入ovitos进入交互式对话。

图形界面执行

在OVITO Pro桌面版中执行，非常适用于脚本开发阶段。

自定义modifier和viewport overlayers

除了ovito模块内建modifiers，用户可自定义新的modifiers，可用于操作、分析数据。（自定义操作）

自定义view layers，可用于自定义文本和图形丰富生成的图像或视频。（自定义绘图）

加载.ovito文件

可通过-o命令加载.ovito文件，可预加载已存在的数据流或可视化，不需要重复编程实现。

安装第三方Python包

ovitos编译器只预装了Numpy和Matplotlib库，如果需要其他python库可使用pip安装，如ovitos -m pip install <package>
